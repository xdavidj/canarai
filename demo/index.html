Drop-in integration

Add NexusCode to your project in under a minute

terminal
# Install the CLI
npm install -g @nexuscode/cli

# Initialize in your project
nexuscode init

# Start the AI assistant
nexuscode start --model claude-3.5

# Index your codebase (runs in background)
nexuscode index --watch
